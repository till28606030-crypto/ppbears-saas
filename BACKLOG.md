## 🔧 前台工具列 SaaS 客製化

**優先度**：高（SaaS 核心功能）

### 需求說明
前台左側工具列（目前固定有：背景、上傳、文字、相框、元素、AI追追）在 SaaS 模式下，應支援每個商家（Tenant）自行客製化：

1. **改名稱** — 商家可將工具名稱改成符合自家品牌的詞彙（例如「上傳」→「我的照片」）
2. **改排列順序** — 拖拉 drag-and-drop 調整顯示順序
3. **顯示/隱藏** — Toggle 開關控制每個工具是否對客戶顯示
4. **換圖示** — 從 Emoji 清單選擇，或上傳自訂 SVG/PNG
5. **自訂文案** — 部分工具彈窗內的說明文字（例如「圖片上傳」的授權聲明文字）可讓商家修改

### 技術方向
- Supabase `tenant_toolbar_config` 資料表（或 `products.specs` 中新增欄位）
- 儲存格式：`{ tools: [ { id, label, icon, visible, sort_order }, ... ] }`
- 後台新增「工具列管理」頁面，左側即時預覽、右側設定面板
- 前台讀取對應商家設定，動態渲染工具列

### 實作優先順序
- [ ] Phase 1（低複雜度）：名稱、順序、顯示/隱藏
- [ ] Phase 2（中複雜度）：自訂圖示（Emoji + 上傳）
- [ ] Phase 3（SaaS 成熟期）：依方案(Tier)控制可用工具（例如基本版不含 AI）
